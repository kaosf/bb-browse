# Browser open shortcut for Bitbucket

## Installation

```sh
# clone this repository
cp bb-browse/bb-browse $HOME/bin # or another directory which is included in $PATH
```

## Usage

```sh
bb-browse
```

You can specify an alias for a remote repository.

```sh
bb-browse origin
```
